# HW5

## Общее

1. Откройте проект в IDEA как Maven-проект
2. Запустите в локальном Tomcat с корневым контекстом `/`
3. Убедитесь в работоспособности:
   - Должны отображаться страницы, похожие на Codeforces
   - На странице `http://localhost:8080/` должна отображаться простая приветственная страница
4. Внимательно прочитайте исходный код проекта (95% этого кода было написано на лекции)

## Задача 1 (обязательная)

***Текущая ситуация:***

- Для всех `action`-методов требуются два обязательных параметра: `HttpServletRequest request`, `Map<String, Object> view`
- В ряде случаев достаточно одного параметра или они вообще не нужны  (например, в `HelpPage` они на самом деле не нужны вообще, а в текущем `IndexPage` нужен только `view`)

***Задание:***

Измените исходный код `FrontServlet` так, чтобы:
- `action`-метод подыскивался просто по названию (без требования к точной сигнатуре)
- При запуске анализировать типы параметров метода (`method.getParameterTypes()`)
- Если параметр имеет тип `Map.class` - передавать как `view`
- Если параметр имеет тип `HttpServletRequest` - передавать соответствующий объект
- Ожидается поддержка любого набора параметров типа `HttpServletRequest` и `Map`, они могут идти в любом порядке
- **!** Ожидается, что ***методы, объявленные в предках/интерфейсах предков, также будут обнаружены*** `FrontServlet`-ом.
  Работу этого пункта необходимо продемонстрировать, создав компонент страницы с предком, реализующим интерфейс с дефолтным методом.
  Показать, что методы предка и дефолтные методы интерфейса корректно находятся.

***Примечание:***

После внесения дополнительного функционала следует:
- Удалить из всех страниц во всех `action`-методах лишние параметры
- Убедиться в работоспособности после изменений

## Задача 2

***Задание:***

По клику на изображения флагов в заголовке сайта осуществляется переход на текущую страницу с параметром `lang=язык`
Вам необходимо реализовать простейшую локализацию на базе этого механизма.

Измените исходный код `FrontServlet` так, чтобы:
   - При наличии параметра `lang` в запросе (он должен содержать две строчные латинские буквы) он запоминался в сессию
   - Если в сессии установлен язык и он отличен от `en`:
     - При загрузке шаблона сначала загрузить шаблон вида `НазваниеСтраницы_язык.ftlh`
     - Если такой шаблон не существует - загружать шаблон без указания языка

***Примечание:***

Обратите внимание, что проект уже содержит IndexPage_ru.ftlh, то есть после реализации решения этой задачи, на главной будет отображаться русская версия этой страницы.

## Задача 3

***Задание:***

По адресу `http://5d3p.wp.codeforces.com/` (пункт меню tic-tac-toe) расположена реализация игры крестики-нолики. Вам необходимо повторить ее функционал у себя в проекте.

***Указания:***

- В проекте есть `TicTacToePage.ftlh` - верстка страницы. Его нельзя изменять, допускаются лишь улучшения (рефакторинг) по сравнению с оригиналом
- Файл `TicTacToePage.java` не содержит реализации логики игры. Все изменения должны производиться здесь
