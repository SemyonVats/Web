# HW10

## Подготовка окружения

[доки реакта](./docs/REACT.md)

##### 1. Установка Node.js
- Установите Node.js с официального сайта: https://nodejs.org/en/
- Рекомендуется использовать последнюю LTS-версию
- Согласитесь на добавление Node.js в PATH во время установки

##### 2. Установка Create React App
- Выполните в командной строке:
  ```bash
  npm install -g create-react-app
  ```
- При использовании Windows рекомендуется использовать cmd.exe вместо PowerShell
- В случае возникновения проблем воспользуйтесь инструкцией: https://gprivate.com/6egq9

##### 3. Создание React-приложения
- Выполните в нужной директории:
  ```bash
  npx create-react-app
  ```
- Будет загружено и настроено необходимое окружение

##### 4. Настройка IDE
- Убедитесь, что используете IDEA с поддержкой React.js
- Или установите WebStorm (в настоящее время доступен бесплатный вариант)

## Подготовка проекта

##### 1. Установка зависимостей
- Перейдите в директорию проекта
- Выполните команду:
  ```bash
  npm install
  ```
- Будут загружены все необходимые модули

##### 2. Настройка запуска
- В IDEA: создайте конфигурацию запуска (Run/Debug Configuration)
- Добавьте новую конфигурацию типа npm
- Параметры:
  - Command: run
  - Script: start
- Альтернативно: запускайте приложение через CLI командой:
  ```bash
  npm run start
  ```

##### 3. Проверка работоспособности
- Убедитесь, что работает:
  - Аутентификация (логины из data.js, пароль не проверяется)
  - Создание постов

## Задача 1: Улучшение Header и Footer
- Для аутентифицированных пользователей выводите в Header:
  - Имя пользователя
  - Логин пользователя
- Добавьте в футер вывод статистики:
  - Количество зарегистрированных пользователей
  - Количество постов
- **!** передавайте только количества, а не массивы объектов

## Задача 2: Реализация регистрации
- Добавьте возможность регистрации новых пользователей
- Изучите механизм добавления постов и реализуйте аналогичную функциональность
- Форма регистрации должна содержать поля:
  - `login` - длина от 3 до 16 символов, только строчные латинские буквы, уникальность
  - `name` - длина от 1 до 32 символов
- После успешной регистрации - перенаправление на страницу входа
- Форма регистрации должна быть визуально похожа на форму входа

## Задача 3: Страница со списком пользователей
- Создайте страницу со списком всех пользователей
- Страница должна быть доступна неаутентифицированным пользователям
- Реализуйте таблицу (`datatable` из прошлых дз) с колонками:
  - `id`
  - `login`
  - `name`

## Задача 4: Главная страница
- На главной странице выводите все посты в порядке убывания id
- Используйте разметку из предыдущих заданий (н-р `article`)
- Отображайте правильное количество комментариев для каждого поста

## Задача 5: Страница отдельного поста
- Реализуйте отдельную страницу для просмотра поста
- Добавьте переходы на эту страницу:
  - По клику на `View all` в сайдбаре
  - По клику на заголовок поста на главной странице
- На странице отображайте:
  - Полное содержимое поста
  - Все комментарии к посту
- **!**: избегайте копипасты кода с главной страницы
- Для комментариев отображайте:
  - Текст комментария
  - Автора комментария

## Задача 6: Добавление комментариев
- На странице поста реализуйте возможность написания комментариев
- Добавьте форму для ввода нового комментария
- Обеспечьте сохранение и отображение новых комментариев
